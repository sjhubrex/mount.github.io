Chapter 1: Structure

Body tag: The purpose of this tag is it holds the content of your webpage you want to be displayed.
The body tag is used on each webpage and contains all of the content of the HTML codes. Everything displayed is within the body tags.

Paragraph tag: This tag will display your content in paragraph form on a webpage. Each new set of a paragraph tag will create a new paragraph with a separation from the previous one.
This tag was used throughout my website to display content in a uniform manner.

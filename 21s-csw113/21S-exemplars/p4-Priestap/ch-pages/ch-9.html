Chapter 9

Chapter 9 showcases even more animation that relates to text fade and scale animation. This is where some elements of the code can begin to become complicated, but with practice and persistence it will be as natural as the other animations previously learned. To break down the animations heading one is set to the "fadeAndScale" with a timing function set and also with a duration time set. The other animation is the scale which is controlled by the keyframes and can be set anywhere from 0-1 and the opacity is also controlled by keyframes.


Overall Throughts

  • Combining text fade and scale animation can be tricky at frist glance, which may make some developers like myself become intimidated by it. However, this new skill provides even more creativity to whatever webpage it is provided on.
  • The continuing of using old knowledge while applying it to new forms of animation is something that I appreciate, since it repeats the basics of animation. This also shows that knowing the basics of CSS and HTML is a must when furthering knowledge of a web developer.

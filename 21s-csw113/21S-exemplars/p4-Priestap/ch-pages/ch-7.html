Since learning how to implement animations and transitions, the more you realize how different these elements actually are. Transitions have a beginning and an end state and should only be used with JavaScript to ensure that they are smooth. Animations are timeless and infinite, which can go on a loop within the time range given. Also, animations much have a keyframe attached to ensure the animation has the right elements.

Chapter 6

The purpose of Chapter 6 was to introduce Transitions, Animations and JavaScript. The main focuse was to learn how to change initial states form using CSS to changes using JavaScript. This is very useful to learn since Java is a very prominent script that is used when coding.


Overall Thoughts

  • Implementing the code and changes for JavaScript was not as difficult as some may think. The changes actually help with how an animation can become smooth and look interesting on a website.
  • A challenge with JavaScript though is trying to understand everything that the code has programed into it. Java is very different than CSS and takes more time to understand what each element means.

MSTC Courses taught by Sarah Huibregtse

Microsoft Office - Introduction

Students should possess basic keyboarding, mouse and Windows XP skills.

Develops introductory skills in the Microsoft Office Suite (Word, Excel, Access, PowerPoint), Windows Explorer, Internet, and computer concepts through demonstrations and lab exercises.

Web Principles: HTML/Expression Web

Corequisite:  Windows Operating Systems or Microsoft Office-Introduction

This course serves two main purposes: (1) it introduces the student to the Internet, including search techniques and email; and (2) it introduces the student to basic web design, including basic XHTML tags and Expression Web.  Students will create and maintain a personal web page, with applications for business web sites.
